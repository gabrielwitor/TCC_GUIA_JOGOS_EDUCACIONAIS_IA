Antes de Começar

Configure o ambiente antes de iniciar as etapas

  1. Baixe e instale o Godot Engine
  2. Baixe e instale o Antigravity
  3. Abra o Antigravity
  4. Crie um novo projeto Godot
  5. Abra a pasta do projeto Godot com o Antigravity
O Antigravity requer conta Google pessoal (não Workspace). Instale antes do workshop para não perder tempo.
SP

Setup do Projeto

Execute este prompt no Antigravity antes de começar as etapas

Com o projeto aberto no Antigravity, cole o prompt abaixo no painel do agente para organizar a estrutura de pastas seguindo as melhores práticas do Godot.

Prompt
Acabei de criar um projeto novo no Godot. Organize a estrutura de pastas seguindo as melhores práticas para a versão mais recente do Godot, considerando que será um jogo 2D educacional simples chamado "Acerte a Sílaba". A estrutura deve incluir pastas para cenas, scripts, assets (imagens e áudio) e fontes. Crie as pastas e explique brevemente para que serve cada uma.
01

Objetivo Pedagógico

Antigravity

Defina o que o jogo vai ensinar, para quem, e com qual embasamento pedagógico. Execute na mesma conversa do Antigravity — o agente vai acumular contexto entre as etapas.

Você também pode usar o Gemini ou ChatGPT se preferir.

Execute os prompts das Etapas 1 a 5 em sequência na mesma conversa do Antigravity. O agente vai acumular contexto e você não precisará repetir informações entre etapas.
Prompt
Estamos desenvolvendo um jogo educacional chamado Acerte a Sílaba. Vou descrever o que quero ensinar com esse jogo:
Com base nisso, crie um documento de objetivo pedagógico estruturado exatamente com: o que ensinar, para quem e com qual método pedagógico.
02

Design do Jogo

Antigravity

Na mesma conversa da etapa anterior, envie o prompt abaixo para gerar o design do jogo.

Você também pode usar o Gemini ou ChatGPT se preferir.

Prompt
Tenho algumas ideias para a mecânica do jogo:
Com base nessas ideias e no objetivo pedagógico que definimos, crie um documento de design do jogo estruturado exatamente com: tipo de jogo, mecânica principal, sistema de feedback e sistema de progressão. Sinta-se livre para sugerir melhorias e complementar o que não ficou definido.
03

Ideação Assistida por IA

Antigravity

Na mesma conversa, peça ao agente para revisar e fortalecer o que foi definido até aqui.

Você também pode usar o Gemini ou ChatGPT se preferir.

Prompt
Revise os dois documentos que criamos — objetivo pedagógico e design do jogo — considerando boas práticas de design de jogos educacionais. Sugira melhorias para fortalecer o aspecto pedagógico, tornar a mecânica mais clara e o feedback mais eficaz.
Leia as sugestões da IA com atenção. Você não precisa aceitar tudo — edite, questione ou peça para ajustar o que não fizer sentido para o seu contexto antes de seguir para a próxima etapa.
04

Design Artístico

Antigravity

Na mesma conversa, peça o documento de design artístico — ele define o estilo visual de todos os assets do jogo.

Você também pode usar o Gemini ou ChatGPT se preferir.

Prompt
Tenho uma ideia visual para o jogo:
Com base nisso e em tudo que definimos até agora, crie o documento de design artístico estruturado com: estilo visual, paleta de cores, padrão dos personagens, padrão dos cenários e tom geral.
Guarde a descrição do cenário e o estilo visual — você vai precisar na Etapa 6.

06

Assets Visuais

Antigravity  +  NanoBanana

Use o Antigravity para gerar os prompts de imagem com base no documento artístico. Depois cole cada prompt no NanoBanana para gerar as imagens.

6a — Imagem de Fundo Antigravity ↗ NanoBanana ↗

Peça ao Antigravity para gerar o prompt. Depois copie o resultado e cole no NanoBanana.

Prompt — Antigravity
Com base no documento de design artístico que criamos, gere um prompt em português para o NanoBanana criar a imagem de fundo do jogo. O prompt deve seguir fielmente o estilo visual, paleta de cores e cenário que definimos. Retorne apenas o prompt, sem explicações.
Copie o prompt gerado pelo Antigravity e cole no NanoBanana para gerar a imagem de fundo.

6b — Imagens das Palavras Antigravity ↗ NanoBanana ↗ remove.bg ↗

Para cada palavra, peça ao Antigravity para gerar o prompt. Depois gere a imagem no NanoBanana e remova o fundo no remove.bg.

Prompt — Antigravity
Com base no documento de design artístico que criamos, gere um prompt em português para o NanoBanana criar a imagem do objeto para o jogo. O prompt deve seguir fielmente o estilo visual que definimos. Retorne apenas o prompt, sem explicações.
Copie o prompt, gere no NanoBanana e remova o fundo no remove.bg. Repita para cada palavra.

6c — Linkar assets ao código Antigravity ↗

Com os assets salvos na pasta correta, peça ao Antigravity para integrá-los ao jogo.

Prompt — Antigravity
Acabei de adicionar os assets visuais na pasta do projeto. Integre as imagens ao jogo. Mantenha o código simples e comentado.
07

Áudio

ElevenLabs  +  Antigravity

Áudio Pedagógico (sílabas e palavras) ElevenLabs ↗

  1. Acesse elevenlabs.io e crie uma conta gratuita
  2. Digite a sílaba ou palavra que deseja gerar (ex: GA, TO, GATO)
  3. Escolha uma voz clara e de tom infantil
  4. Clique em Generate e depois em Download
  5. Repita para cada sílaba e palavra do jogo

7b — Integrar áudios ao jogo Antigravity ↗

Com os áudios gerados, peça ao Antigravity para integrá-los ao jogo.

Prompt — Antigravity
Acabei de gerar os áudios no ElevenLabs. Integre-os ao jogo: use os arquivos de sílabas para o feedback sonoro de cada escolha do jogador, e os arquivos de palavras completas para a pronúncia ao exibir cada palavra na tela. Mantenha o código simples e comentado.
